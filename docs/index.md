# Visual Place Learning Apparatus

This site documents a **low-cost, open-source implementation of a visual place learning arena**, developed in our lab using accessible components and straightforward fabrication methods.

The apparatus is designed to support behavioral experiments based on established visual place learning paradigms, while emphasizing **reproducibility, clarity, and ease of modification**.

---

## Overview

The system integrates thermal control, panoramic visual stimulation, and behavioral tracking to study spatial learning under controlled conditions.  
Design choices prioritize commonly available hardware and simple assembly without altering the core experimental logic.

---

## Scope of Documentation

Here you will find descriptions of:
- The thermal arena and control electronics  
- Visual display and illumination setup  
- Sensing, control, and data acquisition components  

This documentation reflects an ongoing effort and will evolve as the apparatus is refined and used in experiments.
